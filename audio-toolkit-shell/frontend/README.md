# Audio Toolkit Shell - Native Desktop Application

**IMPORTANT: This is a NATIVE macOS DESKTOP APPLICATION built with Tauri, NOT a web browser application.**

This is the frontend for the Audio Toolkit Shell - a native desktop application that provides a multi-terminal dashboard for running 5 audio toolkit launcher executables simultaneously. Built with React + TypeScript + Vite + Tauri.

## Architecture

- **Native Desktop App**: Built with Tauri framework for native macOS desktop experience
- **Frontend**: React + TypeScript + Vite for the UI within the native window
- **Backend**: Rust + Tauri for native process management and system integration
- **Terminal Emulation**: xterm.js for real terminal functionality within the native app

## Development

To run the native desktop application in development mode:

```bash
npm run tauri dev
```

This will open a native macOS desktop window (NOT a web browser) with the multi-terminal dashboard.
